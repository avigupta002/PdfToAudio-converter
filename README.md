# AI PDF Reader & Learning Tool

This project allows users to upload a PDF file and learn its content **line by line** using AI assistance.  
The system converts the PDF into readable text and helps users understand each line step by step.

## Features
- Upload PDF files easily
- AI reads and explains content line by line
- Simple button-based learning flow
- Beginner-friendly interface
- Useful for notes, exams, and self-study

## How It Works
1. Upload your PDF file
2. AI processes and converts the PDF
3. Learn the content step by step using controls

## Buttons Explanation
- **Start** → Begin AI reading from the first line  
- **Next / Push** → Move to the next line or concept  
- **Previous** → Go back to the previous line  
- **Stop** → Pause AI reading  

## Use Case
- Students learning from notes
- Beginners understanding technical PDFs
- Self-paced AI-assisted study

## Technology
- PDF Parsing
- AI Text Processing
- Simple UI Controls

## Goal
Make PDF learning **easy, interactive, and AI-powered**.
